# 🌦 Weather Analytics Dashboard

A full-stack weather dashboard with real-time API data and Power BI visualization.

## 🚀 Features
- Real-time weather data (OpenWeather API)
- Interactive charts (Chart.js)
- Power BI dashboard integration
- Live deployment (Render + Netlify)

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Visualization: Power BI
- Deployment: Render, Netlify

## 🌐 Live Links
- Frontend: https://splendid-khapse-2f1978.netlify.app
- Backend: https://live-weather-forecasting-dashboard.onrender.com

## 📊 Power BI Dashboard
- KPI Cards (Avg, Max, Min Temp)
- Temperature trend
- Humidity analysis
- Weather distribution
